# Voice-to-Text Typo Resolution — NSPFRNP Protocol

**Protocol:** NSPFRNP  
**Type:** Input normalization · Chairman voice-to-text

## Rule

When input is identified or stated as **voice-to-text**:

1. **Expect** typos, homophones, and ASR slips (e.g. Biblandia → Vibelandia, Blandi Arino → Vibelandia Reno, Synthiverse → Syntheverse, intoxicant → intoxication).
2. **Resolve** using canonical forms from repository branding and protocols:
   - **Vibelandia** (not Biblandia, Blandia, Viblandia)
   - **VIBELANDIA RENO!** (not "by Blandi Arino")
   - **Syntheverse** (not Synthiverse)
   - **Mark Twain's Post-Singularity Vibelandia Reno** (canonical)
   - **Hero Harry Houdini** / **Hero Harry** (canonical)
   - **Ben Franklin** (ambassador to Baller Vs)
3. **Preserve** intent and tone. Do not alter meaning; normalize only spelling and canonical names.
4. **Capture** raw intent in review/recording docs when useful; deliver crystallized output with typos resolved.
