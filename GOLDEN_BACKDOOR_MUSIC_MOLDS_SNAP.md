## CANONICAL MUSIC MOLDS

### 1. Vibe / Movement Types

| Mold | Use | Example |
|------|-----|---------|

### 2. Language

| Mold | Rule |
|------|------|
| **Spanglish** | All songs in Spanglish on Golden Backdoor label. Mix Spanish + English; hook in both. |
| **Songs = tags and brands** | Every song is a T-shirt level tag and brand. Wearable, merch-ready. |
| **SNAP SNAP SNAP** | Always layer SNAP SNAP SNAP. Double snap — lock it in. |

### 3. Structure (Simple Repeating)

| Element | Mold |
|---------|------|
| **INTRO** | Build (8–16 bars) |
| **LOOP 1 (main)** | Hook × 4 |
| **LOOP 2 (double)** | Same hook, doubled |
| **BREAKDOWN** | Strip to kick + hook phrase |
| **LOOP 3 (drop)** | Full drop; bring everything back |
| **OUTRO** | Fade on hook |

### 4. Hook Pattern

- **Line 1:** Main phrase (Spanglish)
- **Line 2:** Punch / repeat or translation
- **Line 3:** Phrase broken — dash — dash
- **Line 4:** Vibe tag (e.g. "Pump the air — trance anthem" or "Touchdown vibe — bang — Golden Backdoor")

### 5. Beats Spec

- **BPM:** 124–132 (tribal/downbeat 124–128; trance/progressive 128–132)
- **Elements:** Four-on-the-floor · airy synths · sub bass · open hi-hats on offbeats · build-ups and drops on the hook
- **Drop:** Full bass + kick + main phrase; strip before drop, then bring back on punch line
